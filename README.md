# AI Firewall Support Chatbot

This project is an internal chatbot built using FastAPI and a custom HTML UI.

## Features
- CSV-based FAQ system
- AI-powered semantic search (Embeddings)
- Custom UI rendering
- Clickable links (ServiceNow, Teams)
- Network deployment using IP

## Tech Stack
- Python (FastAPI)
- HTML + JavaScript
- OpenAI API (Embeddings)

## How to Run
python3 -m uvicorn main:app --host 0.0.0.0 --reload
